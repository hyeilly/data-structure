#### 피보나치 수열 1: 이해

- 피보나치 수열
- 피보나치 수열의 구성
  - 수열의 n번째 값 = 수열의 n - 1번째 값 + 수열의 n - 2 번째 값
- 피보나치 수열의 표현
  - fib(n) = 0 ... n = 1
  - fib(n) = 1 ... n = 2
  - fib(n) = fib(n - 1) + fib(n - 2) ... otherwise

### 이진 탐색 알고리즘의 재귀구현1

#### 이진 탐색의 알고리즘의 핵심

1. 탐색 범위의 중앙에 목표 값이 저장되었는지 확인
2. 저장되지 않았다면 탐색 범위를 반으로 줄여서 다시 탐색 시작

#### 이진 탐색의 종료에 대한 논의

- -1의 반환은 탐색의 실패를 의미
