# 그래프

## 그래프의 이해와 종류

- 정점 별로 연결된 간선의 수가 모두 짝수이어야 간선을 한번씩만 지나서 처음 출발했던 정점으로 돌아올 수 있음
- 데이터를 표현하는 방식

1. 무방향 그래프 - 무방향 완전 그래프
2. 방향 그래프 - 방향 완전 그래프
3. 가중치 그래프

- 무방향 가중치 그래프
- 방향 가중치 그래프

### 그래프의 집합 표현

- ex) 무방향 그래프
  - V(G1) = {A, B, C, D}
  - E(G1) = {(A, B), (A, C), (A, D), (B, C), (C, D)}
- 그래프 G 의 정점 집합 `V(G)`
- 그래프 G 의 간선 집합 `E(G)`

- ex) 방향 그래프
  - V(G3) = {A, B, C, D}
  - E(G3) = {<A, B>, <A, C>, <D, A>}
